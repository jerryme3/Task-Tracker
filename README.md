# Task-Tracker
A CLI-based app that tracks users' tasks!

## MADE IN PYTHON

## FEATURES
- Add tasks
- Remove tasks
- Mark tasks as finished
- Mark tasks as unfinished
- See all tasks
- See unfinished tasks only
- See finished tasks only
- See lists of commands

## HOW TO RUN
1. Clone the repository anywhere in your PC
2. Open the terminal within the folder path of the cloned repository
