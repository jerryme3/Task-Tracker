# Task-Tracker
A Python CLI-based Application that tracks users' tasks!

## FEATURES
- Add tasks
- Remove tasks
- Mark tasks as finished
- Mark tasks as unfinished
- See all tasks
- See unfinished tasks
- See finished tasks
- See lists of commands

## COMMANDS
- add      - lets you add a task to the json file. Format: python main.py add "task"
- mark-fin - lets you mark a task as finished. Format: python main.py mark-fin "task-id"
- mark-uf  - lets you mark a task as unfinished. Format: python main.py mark-unf "task-id"
- see-all  - lets you see all of your listed tasks. Format: python main.py see-all
- see-fin  - lets you see all of you finished tasks. Format: python main.py see-fin
- see-uf   - lets you see all of your unfinished tasks. Format: python main.py see-uf
- del      - lets you delete a specific task within your list. Format: python main.py del "task-id"

## HOW TO RUN
1. Clone the repository anywhere in your PC
2. Check if you have the CPython to run this code. If you haven't, download it from their official site, here -> https://www.python.org/downloads. After, verify by running this command: python --version
3. Open the terminal within the folder path of the cloned repository 
4. Run this command to see every possible commands of the app: python main.py help
